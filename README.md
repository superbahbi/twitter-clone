## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/superbahbi/twitter-clone.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run
   ```sh
   npm start
   ```
